## 当前系统痛点分析

### 1. Render 512MB 实例频繁爆内存 (OOM)
**根因分析**：
- PDF解析一次性加载整文件 → 内存峰值过高
- DeepSeek流式响应 + 大上下文消耗
- Redis任务队列 + 异步判卷同时占用内存
- Uvicorn默认workers过多 + Python GC不及时导致内存泄漏

### 2. Supabase 数据交换量巨大、不正常
**根因分析**：
- 反复拉取大字段（申论答案、评分结果等TEXT/JSONB字段）
- 无分页查询导致全量数据往返
- 大JSON数据传输浪费带宽和延迟

#### 1.1.1 Uvicorn 启动命令优化
**文件位置**: Render Dashboard → Start Command

**当前配置**:


**优化后配置**:

**1. PDF解析服务优化**

**示例代码改动**:
```python
# 优化前
def parse_pdf(file_path: str) -> str:
    with open(file_path, 'rb') as f:
        pdf_data = f.read()  # 一次性加载整个PDF
    # 处理逻辑...
    
# 优化后
def parse_pdf_stream(file_path: str, chunk_size: int = 1024*1024) -> str:
    result = []
    with open(file_path, 'rb') as f:
        while chunk := f.read(chunk_size):
            # 处理当前chunk
            processed = process_chunk(chunk)
            result.append(processed)
            del chunk  # 显式释放内存
    return ''.join(result)
```

```python
    async def save_result_to_r2(self, task_id: str, result_data: dict) -> str:

        json_str = json.dumps(result_data, ensure_ascii=False)

        self._save_to_db(task_id, summary, r2_url)

        # 从R2下载
        result_data = await self.cloudflare_r2_service.download_json(r2_url)

    def _extract_summary(self, result_data: dict) -> str:
```

```python
    async def get_user_history_with_cache(self, user_id: int, page: int = 0, limit: int = 20):
```

```python
"""
独立Worker启动脚本
"""
import os
import asyncio
import signal
import sys

from backend.app.workers.task_worker import start_workers, stop_workers

async def main():
    """主函数"""
    print("Starting Shenlun AI Worker...")
    
    # 启动worker
    worker_tasks = await start_workers(num_workers=2)
    
    # 信号处理
    def signal_handler(signum, frame):
        print(f"Received signal {signum}, shutting down...")
        asyncio.create_task(stop_workers(worker_tasks))
    
    signal.signal(signal.SIGINT, signal_handler)
    signal.signal(signal.SIGTERM, signal_handler)
    
    # 保持运行
    try:
        await asyncio.Future()  # 永久运行
    except asyncio.CancelledError:
        print("Worker cancelled")
    finally:
        await stop_workers(worker_tasks)

if __name__ == "__main__":
```

```python
import fitz  # pymupdf

class OptimizedPDFParser:
    def parse_with_stream(self, file_path: str):
        """流式解析PDF"""
        doc = fitz.open(file_path)
        
        try:
            for page_num in range(len(doc)):
                page = doc.load_page(page_num)
                text = page.get_text()
                
                # 立即处理当前页
                processed = self.process_page_text(text)
                yield processed
                
                # 显式释放页面内存
                del page, text
                
                # 每5页触发一次GC
                if page_num % 5 == 0:
                    import gc
                    gc.collect()
        finally:
            doc.close()
```
